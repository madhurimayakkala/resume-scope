# ATS Resume Matcher

A modern AI-powered ATS Resume Matcher built using:

- Next.js
- React
- TypeScript
- Tailwind CSS
- Gemini API
- pdf-parse

## Features

- PDF Resume Upload
- ATS Match Scoring
- Skill Matching
- Missing Skill Detection
- AI-Powered Resume Feedback
- Clean Premium UI
